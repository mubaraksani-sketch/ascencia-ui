# Ascencia UI (React + Vite)

Premium, modern UI for an Ascencia-inspired business school platform. Built with React, Vite, and plain CSS (no UI frameworks).

## Prerequisites

- Node.js (LTS recommended)
- npm (ships with Node)

## Setup

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Run lint:

```bash
npm run lint
```
